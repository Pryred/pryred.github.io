Application Spring de gestion de menus et plats (2025)

Description

Ce projet a impliqué le développement d'une application Spring Boot pour la gestion de menus et de plats dans un contexte de restauration. L'application permet de gérer les relations entre les entités (Plat, Catégorie, Menu) et inclut une interface utilisateur basée sur Thymeleaf ainsi qu'une API REST pour les opérations externes.

Compétences mises en œuvre

  • Développement avec Spring Boot et Java 21
  • Conception et mise en place d'une base de données relationnelle avec MariaDB
  • Implémentation du modèle MVC avec Spring et Thymeleaf
  • Intégration de Bootstrap et Bootswatch pour créer une interface utilisateur responsive et esthétique
  • Création d'une API REST pour les opérations CRUD via clients externes (Postman)
  • Mise en place de filtres et de pagination pour l'affichage des données
  • Implémentation de fonctionnalités comme le calcul automatique des calories totales d'un menu
